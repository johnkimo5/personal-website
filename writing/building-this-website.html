Building this website

September 30, 2025

I decided it was time to refresh my personal website. My goal was to create something minimalist, fast, and easy to maintain.

The Stack

I chose to stick with the basics: semantic HTML and modern CSS. No heavy frameworks, no complex build steps. Just clean code that renders instantly.

Design Philosophy

The design focuses on typography and whitespace. I wanted my work experience and projects to stand out without visual clutter. The single-column layout works perfectly across mobile and desktop devices without requiring complex media queries.

This site is a work in progress, and I'll continue to tweak it as I add more content.
